<header>: Шапка страницы.

<p>: Определяет текстовый абзац.

<nav>: Навигационное меню.

<main>: Основное содержимое страницы.

<section>: Отдельная секция или блок контента.

<article>: Самостоятельная статья или запись блога.

<aside>: Боковая панель или дополнительная информация.

<footer>: Нижний колонтитул или подвал страницы.

<div>: Используется для создания блочных контейнеров и группировки элементов на странице.

<a>: Используется для создания ссылок на другие веб-страницы, документы или разделы на той же странице.

<img>: Тег для вставки изображений на веб-страницу.

<ul> и <li>: <ul> (unordered list) используется для создания маркированного списка, а <li> (list item) для создания элементов списка.

<h1> - <h6>: Используются для заголовков различных уровней. <h1> обычно используется для наиболее значимых заголовков, а <h6> - для менее важных.

<table>: Используется для создания таблиц на веб-странице.

<form>: Используется для создания форм, позволяющих пользователям отправлять данные на сервер.

<input>: Используется внутри тега <form> для создания полей ввода, таких как текстовые поля, чекбоксы, радиокнопки и другие.

<button>: Тег для создания кнопок, которые пользователи могут нажимать для выполнения определенных действий.

<span>: Используется для выделения отдельных частей текста и применения к ним стилей.
